Ocarina makes use of certain HTML elements and CSS properties that require the use of the HTML5 doctype. Include it at the beginning of all your pages.

# Box-Sizing

We reset box-sizing to border-box for every element in Ocarina. This allows us to more easily assign widths to elements that also have padding and borders.

# Normalize

For improved cross-browser rendering, we use Normalize.css to correct small inconsistencies across browsers and devices.
